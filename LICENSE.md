## 🔧 Common Issues and Fixes

### ❌ Injection Fails / "Failed to Attach" Error
**Cause:** Roblox updated after your current RC7 version was compiled, or antivirus is intercepting the loader process.
**Fix:** Restart RC7 — the auto-updater will pull the latest patch on launch. If still failing, add the RC7 folder to your antivirus exclusion list and retry as Administrator.

---

### ❌ Nothing Happens After Clicking Execute
**Cause:** The specific script you're running may be outdated, game-patched, or incompatible with the current Roblox API.
**Fix:** Test a simple script from the built-in Script Hub (e.g., a speed or jump script) to confirm RC7 itself is working. If Hub scripts execute normally, the issue is with the individual script, not the executor.

---

### ❌ RC7 Crashes on Launch (Windows)
**Cause:** Missing .NET 6.0 runtime or outdated Visual C++ Redistributable packages.
**Fix:** Download and install the latest [.NET Runtime](https://dotnet.microsoft.com/download) and Microsoft Visual C++ Redistributable 2022 (x64), then relaunch as Administrator.

---

### ❌ Android APK Won't Install
**Cause:** "Install Unknown Apps" permission not enabled, or insufficient device storage.
**Fix:** Go to Settings → Security → enable "Install Unknown Apps" for your file manager or browser. Ensure at least 250 MB of free internal storage is available before installing the **rc7 executor apk latest version**.

---

### ❌ iOS Build Won't Open / "Untrusted Developer" Error
**Cause:** The developer profile has not been trusted on your device, or the sideloaded IPA has expired.
**Fix:** Settings → General → VPN & Device Management → find the RC7 profile → tap **Trust**. If it has been more than 7 days since sideloading, repeat the **rc7 executor ios install** process using Sideloadly or AltStore.

---

### ❌ Auto-Update Not Working
**Cause:** Outbound connections to RC7 update servers are blocked by a firewall, corporate network, or VPN.
**Fix:** Add `RC7Loader.exe` to your firewall's outbound allow list. Switching to a mobile hotspot temporarily can confirm whether the issue is network-related.

---

### ❌ Script Hub Not Loading / Showing Blank
**Cause:** No internet connection, or Hub servers temporarily down for maintenance.
**Fix:** Check your internet connection first. If online but Hub is blank, wait 5–10 minutes and refresh — Hub updates during Script Hub server maintenance windows can cause brief outages.
